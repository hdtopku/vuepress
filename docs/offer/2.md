📄 题目描述

> 请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为 We Are Happy.则经过替换之后的字符串为 We%20Are%20Happy。

💡 思路

> 遍历字符串，遇到空格的替换拼接上%20，否则，直接拼接

```python
# -*- coding:utf-8 -*-
class Solution:
    # s 源字符串
    def replaceSpace(self, s):
        # write code here
        s1 = ''
        for i in s:
            if i == ' ':
                s1 += '%20'
            else:
                s1 += i
        return s1
```

```php
<?php
function replaceSpace($str)
{
    // write code here
    $str1 = '';
    for ($i = 0; $i < strlen($str); $i++) {
        if ($str[$i] == ' ') {
            $str1 .= '%20';
        } else {
            $str1 .= $str[$i];
        }
    }
    return $str1;
}
```

```javascript
function replaceSpace(str) {
  // write code here
  var str1 = "";
  for (let i = 0; i < str.length; i++) {
    if (str.charAt(i) === " ") {
      str1 += "%20";
    } else {
      str1 += str.charAt(i);
    }
  }
  return str1;
}
```
